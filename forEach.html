método forEach()

O método chama uma função uma vez para cada elemento do array

Observe o código a seguir...

        const numeros = [45, 4, 9, 16, 25];
        let text = "";
        numeros.forEach(value, index, array){
            text += value * 2;
        }

Executando este código no console:

console.log(value) - values: 45, 4, 9, 16, 25;
console.log(index) - índices: 0, 1, 2, 3, 4;
console.log(array) - nome do array: numeros;
